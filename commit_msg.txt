feat(diagnostics): 引入网络流量统计与终端渲染指标监控

Go / PC Agent 侧：
- 在 terminalsession.Runtime 中新增对 PTY 输出事件和字节数（ptyOutputEvents, ptyOutputBytes）的原子统计。
- 在 terminalChannelRuntime 中引入 ScreenWireSnapshot 指标，支持按 snapshot, patch, historyPage 和 other 分类统计屏幕协议帧的发送帧数及字节大小。
- 在 server.go 和 manager.go 中暴露上述运行指标。
- 新增并编写了 terminal_channel_runtime_test.go 与 runtime_test.go 相关的单元测试以验证数据累加正确性。

Android 客户端侧：
- 引入 UidTrafficTracker，利用 TrafficStats API 监控应用 UID 级别的网络带宽收发字节差值。
- 在 WebSocketMuxTransport 中增加帧数及字节数统计（rxFrames, rxBytes, txFrames, txBytes），实现多路复用传输层的吞吐量监控。
- 扩展 TerminalRenderMetrics，引入终端 UI 渲染耗时指标统计。
- 在 DiagnosticLogExporter 中整合了上述流量与渲染度量，支持日志导出分析。
